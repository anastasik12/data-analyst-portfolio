# Концептуальная модель — описание сущностей

## Нормативно-справочные сущности (НСИ)
- Категория товара
- Должность
- Уровень скидочной карты

## Полный список сущностей

| Наименование сущности | Описание |
|-----------------------|----------|
| Товар | Содержит информацию о товарах, которые продает магазин одежды |
| Категория товара | Информация о возможных категориях товаров (юбка, платье, брюки, шорты и т.д.) |
| Поставщик | Хранит информацию о поставщиках, с которыми сотрудничает организация |
| Магазин | Информация о магазинах одной сети |
| Склад | Данные о действующих складах организации |
| Сотрудник | Работники организации, участвующие в продаже товаров |
| Должность | Список должностей |
| Документ | Информация о документах (чеки, накладные, отчёты) |
| Заказ | Информация о заказах клиентов |
| Отзыв | Рейтинги товаров |
| Клиент | Информация о клиентах |
| Адрес | Адреса клиентов |
| Скидочная карта | Скидки, накопленные бонусные баллы |
| Уровень скидочной карты | Справочник возможных скидок |
| Чат поддержки | Чаты клиентов с сотрудниками |
| Сообщение | Текст сообщений в чатах |
| Транзакция | Банковские операции |
| Брак | Бракованные товары, списанные из магазина |
